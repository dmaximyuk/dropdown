### Installation and Local Run

1. Clone the repository:

   ```bash
   git clone https://github.com/dmaximyuk/dropdown
   cd dropdown
   ```

2. Install dependencies:

   ```bash
   npm i
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open the application in your browser at:
   ```
   http://localhost:3000
   ```
